<h1 align="center">Maki</h1>

<h2 align="center">轻量 Markdown 阅读与写作应用</h2>

***

Maki 让你打开 `.md` 文件就能读，点哪里就能改——没有预览与源码的分屏，阅读与写作在同一块画布上完成。

***

## 为什么选择 Maki

Maki 为「先读、再写」而设计:界面克制、干扰少，把注意力留给文字本身。整理笔记、阅读文档、随手记录，都可以像翻书一样打开文件，像在本子上写字一样直接编辑。

如果你习惯 Typora 那种所见即所得的体验，上手会很容易。

***

## 主要功能

### 所见即所得的 Markdown

* 支持标题、列表、任务清单、表格、代码块、引用、链接、图片等常用语法

* 点击正文任意位置即可编辑，无需在预览与源码之间切换

### 舒适的阅读体验

* 内容居中排版，留白充足，适合长时间阅读

* 浅色 / 深色主题一键切换

* 阅读缩放可调，按自己的习惯看清文字

### 本地文件，随手可用

* 打开、保存、另存为，文件始终保存在你的电脑上

* 将 `.md` 文件拖入窗口即可打开

* 侧边栏记录最近文件，方便继续上次的工作

* 自动生成目录大纲，长文也能快速跳转

### 贴心细节

* 关闭前提醒未保存的更改，避免误丢内容

* 侧边栏支持搜索与多选管理最近文件

* 欢迎页背景可自定义，让空状态也有一点趣味

* 常用快捷键可在设置中查看，并支持自定义

***

## 快速开始

1. **打开文件**:点击欢迎页的「打开 Markdown 文件」，或将 `.md` 文件拖入窗口；也可按 `Ctrl + O`
2. **开始写作**:在正文中直接点击并输入
3. **保存**:`Ctrl + S` 保存，`Ctrl + Shift + S` 另存为
4. **查看目录**:打开侧边栏，切换到「目录」，按标题层级浏览文档结构
5. **切换主题**:按 `Ctrl + Shift + L`，或点击界面上的主题按钮

***

## 常用快捷键

| 快捷键                | 作用          |
| ------------------ | ----------- |
| `Ctrl + O`         | 打开文件        |
| `Ctrl + S`         | 保存          |
| `Ctrl + Shift + S` | 另存为         |
| `Ctrl + \`         | 显示 / 隐藏侧边栏  |
| `Ctrl + Shift + L` | 切换浅色 / 深色主题 |

更多快捷键可在应用内「设置 → 热键」中查看与修改。

***

## 设为默认打开方式

安装后，可将 Maki 设为 `.md` 文件的默认打开方式，之后双击即可直接进入:

1. 在任意 `.md` 文件上右键 →「打开方式」→ 选择 Maki → 勾选「始终使用此应用」
2. 或在应用菜单中选择「设为默认 Markdown 打开方式…」，按提示完成设置

***

## 关于

Maki 专注做好一件事:让 Markdown 用起来简单、好看、顺手。

如有问题或建议，可在应用内「设置 → 关于」查看版本与联系方式，或前往 GitHub 项目页面反馈。
