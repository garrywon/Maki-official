# Maki

**一款轻量的 Markdown 阅读与写作应用。**

打开 `.md` 文件，即可在舒适、居中的版式中阅读；需要修改时，点哪里改哪里——没有预览与源码的分屏，写作与阅读在同一块画布上自然切换。

---

## 为什么选择 Maki

Maki 为「先读、再写」而设计。界面克制、干扰少，让你把注意力留给文字本身。无论是整理笔记、阅读文档，还是随手记录想法，都可以像翻书一样打开文件，像在本子上写字一样直接编辑。

如果你习惯 Typora 那种所见即所得的 Markdown 体验，Maki 会很容易上手。

---

## 主要功能

### 所见即所得的 Markdown

- 支持常用 Markdown 语法：标题、列表、任务清单、表格、代码块、引用、链接、图片等
- 点击正文任意位置即可开始编辑，无需在「预览」和「源码」之间来回切换

### 舒适的阅读体验

- 内容居中排版，留白充足，长时间阅读也不累
- 支持浅色与深色主题，随时切换
- 可调整阅读缩放比例，按自己的习惯看清文字

### 本地文件，随手可用

- 打开、保存、另存为——文件始终保存在你的电脑上
- 支持将 `.md` 文件拖入窗口直接打开
- 侧边栏记录最近打开的文件，方便继续上次的写作
- 文档内自动生成目录大纲，长文也能快速跳转

### 贴心细节

- 关闭前会提醒未保存的更改，避免误丢内容
- 侧边栏可搜索、多选管理最近文件
- 欢迎页背景可自定义（数量、模糊、动效等），让空状态也有一点趣味
- 常用快捷键可在设置中查看，并支持自定义

## 快速开始

1. **打开文件**：点击欢迎页的「打开 Markdown 文件」，或将 `.md` 文件拖入窗口；也可使用快捷键 `Ctrl + O`（macOS 为 `Cmd + O`）
2. **开始写作**：在正文中直接点击并输入
3. **保存**：`Ctrl + S`（macOS 为 `Cmd + S`）；另存为使用 `Ctrl + Shift + S`
4. **查看目录**：打开侧边栏，切换到「目录」标签，按标题层级浏览文档结构
5. **切换主题**：使用 `Ctrl + Shift + L`（macOS 为 `Cmd + Shift + L`），或点击界面上的主题按钮

---

## 常用快捷键

| 快捷键 | 作用 |
|--------|------|
| `Ctrl/Cmd + O` | 打开文件 |
| `Ctrl/Cmd + S` | 保存 |
| `Ctrl/Cmd + Shift + S` | 另存为 |
| `Ctrl/Cmd + \` | 显示 / 隐藏侧边栏 |
| `Ctrl/Cmd + Shift + L` | 切换浅色 / 深色主题 |

更多快捷键可在应用内「设置 → 热键」中查看与修改。

---

## 设为默认打开方式

安装后，可将 Maki 设为系统默认的 Markdown 打开方式，之后双击 `.md` 文件即可直接进入 Maki：

- **Windows**：在任意 `.md` 文件上右键 →「打开方式」→ 选择 Maki → 勾选「始终使用此应用」；也可在应用菜单中选择「设为默认 Markdown 打开方式…」
- **macOS**：在 `.md` 文件「显示简介 → 打开方式」中选择 Maki，并点击「全部更改…」
